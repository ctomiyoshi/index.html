- CONCEPT -

あなたの大切な商品とユーザー様を結びます。 当社の安全安心に特化したドライバーにお任せください。

現在、私共は関西をメインに軽貨物車両で行える全ての物流をお手伝いしております。

配送のアウトソーシングはTOMIYOSHI TRANSPORTへ

と、ここまではごくアリキタリな軽貨物事業の会社が謳っているキャッチフレーズですが。 ハッキリ言わせていただきます。
私共はその辺の軽貨物ドライバーとは一味違います。 大きな違いは『実際に走る』ドライバーの責任感の重みです。
この『実際に走る』と言うニュアンスがどういうものかはクライアント様、ユーザー様そして、ドライバー本人が一番理解していただけているものかと思います。第三者であるドライバーに配送を委託するクライアント様の不安、第三者を経由し届けられるユーザー様の不安、このふたつの不安を解消するにはドライバーの絶対的信用と責任感が必要不可欠です。しかしどうでしょう。大手や規模の大きい物流会社に配送を委託しても実際に走るドライバーは下請けや孫請けのなんの責任感もないドライバー達です。それが故に大切な商品すらも管理が杜撰で、まともに機能していない悲惨な現場をこれまで幾度と見てきました。そういった現場を改善、またはつくらない為にも当社では当社以外のドライバーに配送のお仕事をお願いすることはありません。つまり、外注のご依頼は承っておりますが、当社から更に第三者の配送業者に外注依頼することはありません。『TOMIYOSHI TRANSPORT』という名前を背負った品格と責任感のあるドライバーだけが、あなたの大切な商品を運びます。ドライバー自身の責任感が生まれると必然的にパフォーマンス向上に繋がります。商品の管理の徹底、丁寧な運転と対応、高度なパフォーマンスを自発的に向上させ、クライアント様とユーザー様の信頼関係の更なる潤滑油となります。パフォーマンスが低下しないよう、ドライバーの体調管理も徹底しております。同業者のドライバーで『稼ぎたいから』と言う理由でハードに稼働しているドライバーを何人も見てきましたが結末は『事故を起こしてしまった』か『心か身体を壊して仕事ができなくなってしまった』の決まってどちらかです。これでは話になりません。ドライバーの負担が大きくなってしまう理由は、そのドライバーが所属している運送会社にほとんどあるかと考えております。当社はドライバーに過度な負担がかからないように料金設定し、その都度スケジュール調節させていただいております。そもそもドライバーに過度な負担がかかるお仕事はお断りさせていただいておりますのでご了承下さい。
